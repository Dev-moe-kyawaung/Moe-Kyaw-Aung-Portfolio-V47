## 🎯 Overview

**Moe-Kyaw-Aung Portfolio V47** is a professional, production-ready portfolio website designed for Android developers and tech professionals. Built with a **brutalist aesthetic** featuring **Electric Orange accents (#e8590c)** and a **dark minimalist interface**, this portfolio combines stunning visual design with exceptional functionality.

### Core Concept
This portfolio serves as a comprehensive digital presence showcasing:
- **12+ years** of Android development expertise
- **50+ completed projects** across various domains
- **Specialized skills** in Kotlin, Jetpack Compose, Firebase, and clean architecture
- **Professional services** and pricing packages
- **Global presence** with multiple portfolio platforms
- **Social proof** through testimonials and certifications

### Target Audience
- Potential clients seeking senior Android developers
- Hiring managers and recruiters
- Freelance project collaborators
- Technical mentors and educational institutions
- Industry peers and tech community members

## 🚀 Installation

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor (VS Code, Sublime, Atom, etc.) - for customization
- Git (for cloning the repository)
- Optional: Node.js (for local development server)

### Step 1: Clone the Repository

```bash
# Clone via HTTPS
git clone https://github.com/moekyawaung-tech/portfolio-v47.git

# Or clone via SSH
git clone git@github.com:moekyawaung-tech/portfolio-v47.git

# Navigate to project directory
cd portfolio-v47
```

### Step 2: Install Dependencies (Optional)

For local development with a simple HTTP server:

```bash
# Using Python 3 (comes pre-installed on most systems)
python -m http.server 8000

# Or using Node.js (if installed)
npx http-server

# Or using Ruby
ruby -run -ehttpd . -p8000
```

### Step 3: Open in Browser

```bash
# If using Python server
open http://localhost:8000

# Direct file opening (less preferred)
open index.html
```

### Step 4: Customize (Optional)

Edit the HTML file to personalize:

```bash
# Open in your editor
code index.html      # VS Code
vim index.html       # Vim
nano index.html      # Nano
```

Refer to [CUSTOMIZATION.md](./CUSTOMIZATION.md) for detailed customization instructions.

## ⚡ Quick Start

### For First-Time Users

1. **Clone & Open**
   ```bash
   git clone https://github.com/moekyawaung-tech/portfolio-v47.git
   cd portfolio-v47
   open index.html
   ```

2. **Basic Customization**
   - Replace "Moe-Kyaw-Aung" with your name
   - Update the tagline and bio
   - Add your projects and images
   - Update contact information

3. **Deploy**
   ```bash
   # Push to GitHub
   git add .
   git commit -m "Customize portfolio"
   git push origin main
   
   # Enable GitHub Pages in repository settings
   ```

### For Developers

1. **Development Setup**
   ```bash
   # Clone repository
   git clone <your-fork-url>
   cd portfolio-v47
   
   # Start local server
   python -m http.server 8000
   
   # Open browser
   open http://localhost:8000
   ```

2. **File Editing**
   - Edit `index.html` for HTML structure
   - Modify `:root` CSS variables for theming
   - Update JavaScript functions for behavior changes

3. **Testing**
   - Test responsive design (DevTools F12)
   - Verify all links and forms work
   - Check animations in different browsers

4. **Deployment**
   - See [DEPLOYMENT.md](./DEPLOYMENT.md) for detailed instructions
   - Supports GitHub Pages, Netlify, Vercel, and more

## 🎨 Customization

### Quick Color Changes

Edit CSS variables in the HTML `<style>` section:

```css
:root {
    --color-primary: #e8590c;        /* Orange accent */
    --color-dark: #1a1a1a;           /* Dark background */
    --color-warm: #FFF7ED;           /* Light accent */
    --color-text-dark: #333333;      /* Dark text */
    --color-text-light: #f5f5f5;     /* Light text */
}
```

## 🚀 Deployment

### Quick Deploy to GitHub Pages

```bash
# 1. Ensure you're on main branch
git checkout main

# 2. Create gh-pages branch if not exists
git checkout --orphan gh-pages
git rm -rf .

# 3. Copy only index.html
cp ../main/index.html .
git add index.html
git commit -m "Deploy portfolio to GitHub Pages"
git push -u origin gh-pages

# 4. Return to main
git checkout main
```

### Deploy to Netlify

```bash
# 1. Connect GitHub repository to Netlify
# 2. Set build settings:
#    - Build command: (leave empty)
#    - Publish directory: . (root)
# 3. Deploy with one click
```

### Deploy to Vercel

```bash
# 1. Install Vercel CLI
npm install -g vercel

# 2. Deploy
vercel

# 3. Follow prompts to connect to GitHub
```

### Deploy to Traditional Hosting

```bash
# 1. Upload index.html via FTP/SFTP to your server
# 2. Set proper permissions (644 for files)
# 3. Access via your domain
```
